## 환경 레코드

환경 레코드는 어휘적 중첩 구조를 기반으로 식별자와 변수 혹은 함수의 연관 관계를 정의하는 명세 타입이다. 함수 선언문, 블록문, catch문과 같은 문법 구조와 연관이 있다. 코드가 평가될 때 환경 레코드가 생성된다. 모든 환경 레코드는 `[[OuterEnv]]`라는 내부 슬롯을 가지며, 이 슬롯은 외부 환경 레코드 대한 참조 혹은 null 값을 가진다. 환경 레코드는 선언적 환경 레코드, 객체 환경 레코드, 전역 환경 레코드의 추상 클래스이다. 함수 환경 레코드와 모듈 환경 레코드는 선언적 환경 레코드를 상속받는다.
